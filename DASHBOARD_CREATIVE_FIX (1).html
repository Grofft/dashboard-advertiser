Integrasi Spreadsheet (Google Apps Script)

Tarik & kirim data otomatis ke Google Sheets
Auto-Sync Aktif: setiap kali halaman dibuka, dashboard otomatis menarik data terbaru dari spreadsheet (asal URL Apps Script sudah diisi).
Belum disinkronkan.
Buka Google Sheet Anda → Extensions → Apps Script → tempel kode di bawah → Deploy → New deployment → Web app (Execute as: Me, Who has access: Anyone) → salin URL hasil deploy ke kolom di atas. Buat 2 sheet bernama Produksi (kolom: tanggal, judul, nama, tipe, platform, views, likes, comments, shares) dan Akun (kolom: tanggal, nama, platform, status, followers, note, salesUnit, salesOmzet). Sheet RequestKonten (kolom: tanggal, judul, requester, pic, produk, platform, deadline, prioritas, status, brief) akan dibuat otomatis saat request konten pertama dikirim.
